# M.C-plusOuMoins

Créer un jeu de devinette simple où l'utilisateur doit deviner un nombre secret en saisissant des chiffres dans un champ de saisie. Le jeu doit fournir des indices tels que "plus grand", "plus petit" ou "égale" pour aider l'utilisateur à trouver le nombre secret.

### General Info

---

**Fonctionnalités :**

- Un champ de saisie pour que l'utilisateur entre un nombre.
- Un bouton pour soumettre la tentative de l'utilisateur.
- Un message indiquant si le nombre deviné est trop grand, trop petit ou correct.
- Un message de félicitations lorsque l'utilisateur trouve le nombre secret. Avec des feux d'art
